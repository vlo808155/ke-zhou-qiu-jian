# 冰清玉润

> 拼音：bīng qīng yù rùn

## 成语释义

润滋润。象冰一样晶莹，如玉一般润泽。原指晋乐广卫玠翁婿俩操行洁白◇常比喻人的品格高洁。

## 成语出处

南朝·宋·刘义庆《世说新语·言语》刘孝标注引《卫玠别传》裴叔道曰妻父有冰清之姿，婿有璧润之望。”

## 使用示例

他是～，怎便肯随波逐尘。★明·高濂《玉簪记》第三十出
